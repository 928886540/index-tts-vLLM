声腔参考音频说明

来源：
- 声腔素材/audio [vocals].wav

用途：
- 这些文件是给 IndexTTS2 的 emo_audio_prompt / style 参考用的，不是普通角色音色。
- 后端会通过 style 名称读取 prompts/library/声腔/{style}.wav。
- LLM 只需要输出 style 字段，例如 "style": "scream_peak"，后端会自动映射到对应 wav。

强度建议：
- neutral：不用参考音频。
- 轻声/贴耳：style_alpha 0.36-0.48。
- 重喘/升温：style_alpha 0.50-0.58。
- 尖叫/高潮峰值：style_alpha 0.60-0.66，不要再高，太高容易噪音和破音。
- moan_soft、scream_peak 不适合长台词，只适合短促反应、断句、拟声或峰值片段。

阶段映射：
- stage_warmup -> breath_soft
- stage_rising -> intimate_breath
- stage_peak -> scream_peak
- stage_afterglow -> low_murmur

常用判断：
- 前戏/刚开始：breath_soft、whisper_soft、shy_whisper。
- 升温/呼吸变重：intimate_breath、tense_breath、breath_heavy。
- 短促反应/呻吟：gasp_surprise、moan_soft。
- 高潮/尖叫：scream_peak 或 stage_peak。
- 余韵/事后低声：low_murmur 或 stage_afterglow。

中文同名声腔：
- 命名格式：{声腔}-{人名}.MP3，例如 喘息-步非烟.MP3、喘息-AD学姐.MP3、喘息-JOK.MP3。
- 已有人名：步非烟、AD学姐、JOK。
- 已有声腔：轻喘、喘息、耳语、低语、低吟、惊喘、哭腔、哽咽、挑逗、轻笑、尖叫、余韵。
- LLM 可直接把 style 写成不带扩展名的中文 ID，例如 "style": "喘息-AD学姐"。
- 后端会映射到 prompts/library/声腔/{style}.MP3。
